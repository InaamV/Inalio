Réalisation : Tinycar

Description 📖:

Cette réalisation a été faite seule. Les consignes étaient sur le site du proffesseur M.Frebourg. Il fallait réaliser des programmes en java répondant aux consignes. Cela coprenait par exemple des entrées utilisateurs concernant différents facteurs d'une voiture souhaité

Objectifs 🎯:

L'objectif était de rendre sur gitlab les missions de Tinycar avec une contrainte de temps pour chacune d'elle. Ces missions permettaient aussi de s'améliorer en language Java.

Outils 🛠️:

  • La réalisation à entièrement été faite en Java.
  • Vscode a été l'IDE utilisé pour ce projet.
  • Gitlab - Pour rendre le trvail.

Avis ⏳:

J'ai trouvé que Tinycar était une bonne introduction au language Java, avec les entrée utilisateurs, l'utilisation de tableaux etc..
