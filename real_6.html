Réalisation : Rugby

Description 📖:

Realisation d'un site web en Django présentant les prochains évenements du rugby XV au japon en se basant sur une api donnée. La génération des billets faisait aussi parti de la réalisation. Projet réalisé seul.

Objectifs 🎯:

L'objectif est de s'ameliorer en Django, html, css et en utilisation d'Api.

Outils 🛠️:

  • Vscode
  • github
  • Django
  • Python
  • html
  • css

Ce que je ferais donc différement, serais de commencer par avoir une application fonctionnelle, puis de commencer à faire les détails pour rester dans les temps.
