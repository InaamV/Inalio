Réalisation : TextAdventure

Description 📖:

Ce projet nous à été demander par Michaël Hubert. Il consiste à créer à partir d'un programme de base fourni, un jeu se jouant dans la console où nous représentons un héro dans le lycée A.U. Nous devons pouvoir nous y déplacer et effectué des actions. Le projet était à faire en Python

Objectifs 🎯:

L'objectif avec cette réalisation est de s'améliorer sur le language Python tout en créant un jeu à nous goût et de manière divertissante. Le projet était à réaliser seul et avec une limite de temps.

Outils 🛠️:

  • Python
  • Vscode
